# Is My Data Still There?

A public information resource on digital storage lifespan and format obsolescence, built as a webpage project for LIS 5268: Technology for Information Professionals at the University of South Florida's School of Information.

**Live site:** https://atruesteele.github.io/is-my-data-still-there

**What's on it:**
- Informational sections on optical, magnetic, flash, and cloud storage types
- A format obsolescence timeline from the 1970s to now
- An "Important — Don't Wait!" callout on the urgency of data backup
- An interactive data lifespan calculator
- A 3-2-1 backup rule guide
- Verified sources in APA 7 format

**Accessibility features:**
- Font switcher with three options (Y2K, Readable, High Legibility)
- Browser zoom instructions for users who need larger text
